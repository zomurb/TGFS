# 🚀 TGFS — Telegram File System

**TGFS** — это мощный CLI-инструмент, превращающий ваш Telegram в безлимитное, безопасное и сверхбыстрое облачное хранилище. Ваши файлы разбиваются на части, шифруются по стандарту AES-256 и загружаются в облако Telegram.

---

## ✨ Основные возможности

- **⚡ Сверхскорость**: Параллельная загрузка и скачивание чанков файла с использованием `asyncio`.
- **🔐 Безопасность**: Полное клиентское шифрование (AES-256). Файлы попадают в Telegram уже в зашифрованном виде.
- **📦 Безлимит**: Храните файлы любого размера (автоматическое разбиение на части).
- **🖥️ Удобный CLI**: Интуитивно понятные команды в стиле SSH/FTP.
- **📊 Индикация**: Красивые прогресс-бары для каждой операции (библиотека `rich`).
- **🔍 Индексация**: Мгновенный поиск и просмотр файлов через локальную базу данных SQLite.

## 🛠️ Установка

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/zomurb/TGFS.git
   cd TGFS
   ```

2. **Установите зависимости:**
   ```bash
   pip install telethon typer cryptography rich colorama
   ```

3. **Настройте API:**
   Отредактируйте файл `config.py` или создайте его, указав свои данные с [my.telegram.org](https://my.telegram.org):
   ```python
   API_ID = 33045341
   API_HASH = "86c1a45367c92315e8a835094613c67c"
   ```

## 🚀 Инструкция по использованию

### 1. Инициализация
Перед первым запуском необходимо создать базу данных:
```bash
python main.py init-fs
```

### 2. Авторизация
При первой загрузке или скачивании Telegram попросит вас ввести номер телефона и код подтверждения. Это создаст файл сессии `tgfs_session.session`. **Никому не передавайте этот файл!**

### 3. Загрузка файлов (Upload)
Загрузить файл в облако:
```bash
python main.py upload "путь/к/файлу.zip"
```
*Файл будет автоматически разбит на части по 48МБ, зашифрован и отправлен в ваши "Избранные сообщения".*

### 4. Просмотр списка файлов (List)
```bash
python main.py ls
```
Вы увидите таблицу с ID, именами, размерами и датами загрузки.

### 5. Скачивание файлов (Download)
Скачать файл по его ID из списка:
```bash
python main.py download [FILE_ID] --output "./downloads"
```

### 6. Поиск файлов
```bash
python main.py search "название"
```

### 7. Удаление файлов
Удалить файл из базы и стереть все сообщения с его частями из Telegram:
```bash
python main.py rm [FILE_ID]
```

### 8. Информация о файле
Посмотреть детальные метаданные (кол-во чанков, хэш, статус шифрования):
```bash
python main.py info [FILE_ID]
```

## 🛡️ Безопасность и Шифрование

TGFS использует библиотеку `cryptography` для реализации симметричного шифрования AES-256 в режиме CBC/GCM. 
- **Ключ шифрования** генерируется локально и хранится в базе данных (в будущем планируется поддержка мастер-пароля).
- Telegram видит только набор случайных байтов.
